# Paaspop Aftelklok (HTML + CSS + JS)

## Wat zit erin?
- index.html  → de structuur (tekst + SVG logo)
- style.css   → alle styling (met uitleg per blok)
- script.js   → de aftel-logica + logo animatie (met uitleg per blok)

## Zo gebruik je het
1. Zet alle bestanden in dezelfde map.
2. Voeg je achtergrondbestand toe:
   - `BackgroundGifPaaspop.gif` (zelfde map)
3. Open `index.html` in je browser.

## Datum aanpassen
Open `script.js` en pas dit aan:
`const EVENT_DATE = new Date(2026, 3, 5);`
Let op: maand is 0-based (April = 3).
